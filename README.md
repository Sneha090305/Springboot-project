# Hello Messaging – Spring Boot REST Application

## Overview
This project is a simple Spring Boot REST API application that demonstrates how to create and run basic REST endpoints. The application returns greeting messages when requests are sent to the server.

This project is created using Spring Boot, Maven, and Java 17.

## Technologies Used
- Java 17
- Spring Boot
- Maven
- Spring Web
- IntelliJ IDEA

## Running the Application
1. Open the project in IntelliJ IDEA
2. Run the main class:
   DependencyInjectionDemoApplication.java
3. The application will start on port 8080

## Testing the API
Open your browser and enter:
http://localhost:8080/hello
